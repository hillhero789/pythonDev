新增：
git init
git add .
git commit -m "say somethin"
git push --set-upstream git@github.com:hillhero789/pythonDev.git master		//git@github.com:hillhero789/pythonDev.git这是远方仓库地址

修改后提交到github.com的方法：
git update  			//检查更新
git add .			//将修改内容加入git	
git commit -m "对提交的注释"	//提交到本地
git push			//提交到github.com
git status			//查看状态
